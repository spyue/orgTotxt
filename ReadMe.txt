Org to txt file that tries to help with formatting 
(right now just replaces * with " ")

Windows 11:
	0. Make sure python 3.x is installed
	1. open terminal 
	2. move to the folder containing the script 
	   use cd to move and ls to see files and folders
	3. type python orgTotxt.py into the terminal *don't hit
	   enter yet*
	4. drag the org file you want to convert onto the 
	   terminal
	5. once you see the file path in the terminal click 
	   enter 
	6. your new txt file will be with the original org file
